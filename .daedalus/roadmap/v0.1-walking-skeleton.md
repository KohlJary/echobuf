# v0.1 — Walking Skeleton

**Status:** ready
**Priority:** P0
**Type:** milestone

## Scope

The minimum viable loop: capture system audio → buffer in RAM → flush to WAV on CLI trigger.

## Work Items

1. **Project scaffolding** — pyproject.toml, package structure, entry point
2. **Ring buffer** — preallocated NumPy circular buffer with write/snapshot
3. **PulseAudio backend** — system capture via `parec` subprocess, float32 PCM chunks
4. **Capture thread** — dedicated thread reading from backend into ring buffer
5. **WAV writer** — snapshot → soundfile.write() to hardcoded output path
6. **CLI entry point** — `echobuf daemon` (foreground) and `echobuf save` (trigger via signal or simple mechanism)
7. **Integration** — wire it all together, test end-to-end
